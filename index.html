Introduction

What is a CSS Grid? CSS Grid 

display: grid;
is designed to help you create organized CSS layouts using a grid system. It's similar to using 
display: flexbox;
except it is two-dimensional.

Why would you use this? Well, almost everything you know about styling a webpage is a hack. Using tables, floats, and a variety of other tricks to layout a page were just hacks to make some things work.

All you need is a container.

<div class="container"> <section class="container"> <article class="container"> Then you just style it with: 
 display: grid
For example: .container { display: grid; } Pretty simple, right?
